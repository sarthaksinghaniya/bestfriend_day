# Best Friend App

A modern React web app built with Vite, Tailwind CSS, Framer Motion, and React Router.

## Features

- React + Vite setup
- Tailwind CSS styling
- Framer Motion animations
- Smooth scrolling enabled
- Google Font: `Poppins`
- Minimal, centered, mobile-responsive UI
- Soft gradient theme with purple, grey, and pink tones

## Pages

1. `Landing`
2. `NameInput`
3. `Letter`
4. `NicknameReveal`

## Folder Structure

```text
src/
  components/
  pages/
  assets/
  utils/
```

## Getting Started

### Install dependencies

```bash
npm install
```

### Run the dev server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- React Router DOM

## Notes

- The app uses a shared layout and route-based navigation.
- The default visual style is tuned for a soft, dreamy presentation.
- You can extend the current pages with form state, animations, or richer content as needed.
